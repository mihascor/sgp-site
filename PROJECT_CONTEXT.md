Проект: сайт-визитка (Next.js)

Технологии:
- Next.js (App Router)
- TypeScript
- CSS Modules

Структура:
- app/ — страницы
- components/ — переиспользуемые компоненты
- sections/ — секции страниц

Особенности:
- без API
- без форм
- без авторизации
- основной упор на верстку по Figma

Правила:
- секции не переиспользуются
- компоненты выносятся только если повторяются
- стили через CSS Modules
- отвечать кратко, пошагово
- Когда в ответе появляется слово или фраза на английском языке в скобках пеши перевод на русский язык. Пример Home (Дом).
